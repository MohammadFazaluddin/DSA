- Arrays (2-D, Multi-Dimensional),
- Linked Lists (Doubly Linked list),
- Sparse Matrices,
- Stacks,
- Queues (Circular Queue, Deque, Priority Queue),
- Trees,
- Graphs

- External Sorting,
